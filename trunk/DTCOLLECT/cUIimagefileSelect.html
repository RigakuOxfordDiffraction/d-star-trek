Image File/Template Selection Dialog

The Image Template dialog is used to specify a unique naming scheme for a series of images from a scan. Enter a template or filter that contains question marks in the top text field of the dialog. Select Filter to see files that match the template.
Starting seq. number
The starting or first sequence number
Increment
The increment between images. This value will be added to the starting sequence number to get the second image number. It may be negative to decrement the sequence number.
Number of images
The number of images to use in operations like Tile, Overlay, Average or Statistics.
Examples

Below are three examples. In each case the template, the starting sequence number, the increment and number of images are given on one line and the resulting image filenames are listing on subsequent lines.

ribosome.???? 1 1 5
ribosome.0001, ribosome.0002, ribosome.0003, ribosome.0004, ribosome.0005

ribosome?.??? 2 2 3
ribosome0.002, ribosome0.004, ribosome0.006

lyso5.??? 5 -1 3
lyso5.005, lyso5.004, lyso5.003
