dtcollect on-line help: Main Window Menubar

File menu

The File menu is used to initialize or re-initialize the hardware devices. The commands in the file menu are Initialize and Exit.

Initialize
The Initialize command popups a menu with the following choices:
  • Crystal goniometer Call initialize routine of the crystal goniometer
  • Detector goniometer Call initialize routine of the detector goniometer
  • Detector Call detector initialize routine
  • Shutter Call shutter initialize routine
  • Sensors Call sensors (counters) initialize routine
  • Source Call source device initialize routine
  • All Call all the hardware device initialize routines
Each of these calls the initialize routine of the respective hardware device. These routines in turn are site and installation dependent.
Image template...
This command is obsolete. Use the Scan Template ... button in the main window instead. This lets you view and edit the current settings for the image filename template and sequence numbers. The image file selection template dialog box appears as shown elsewhere. The settings in the dialog box come from the header of the last image read in or from the last time the image template dialog appeared. The template is set from the top field, NOT the selected filename. The template should contain question mark characters to be used by d*TREK functions.
Exit
This command exits from dtcollect.

Edit menu

The edit menu has commands to display dialog boxes that let you edit the some information that will appear in image header and to clear the output scrolled text window.

Image header items...
This is currently not implemented. It pop-ups an unusable dialog box. Select Cancel or Dismiss in this box to dismiss it.
Clear output
You can clear the output scrolled text window with this item. You have two choices, either clear all the text or only selected text. To clear selected text, first select the text to clear, then press Edit/Clear output/Selected text.

View menu

Commands in the View menu are all desensitized. This menu has no active commands as of now.

Image header...
Use the dtdisplay to view image headers.

Collect menu

The Collect menu provides a way to collect more complicated scans with the dtcollect program. Presently, only one choice is implemented: Scan...

Scan ...
This pop-ups the scan dialog where you can create, edit and save scans. You can also collect a single scan or multiple scans with the scan dialog. See Scan... for details.

Help menu

This help menu provides access to the hyper-text markup language viewer. Context-sensitive help is provided in the main window title whenever a button is armed. (In Motif version 1.2, the title will also provided help on text fields when they gain keyboard focus.)

Help via the HTML viewer
Select this command to activate a hyper-text markup language (HTML) viewer to view the dtcollect HTML help files. The default HTML viewer is NCSA Mosaic, but this may be changed in the resource file. If the viewer is already running in the process tree, then it is used, otherwise a new process is created that runs the viewer.
