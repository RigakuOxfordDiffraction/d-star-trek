dtcollect on-line help

Edit detector items dialog

With the Edit detector items dialog you can enter the detector modes for data collection.

Binning mode
Select the mode for binning pixels during detector readout.
1x1
In 1x1 mode, no binning is done. The full pixel resolution of the detector is used.
2x2
In 2x2 mode, 4 pixels (2 in each direction) are binned during readout. Images are 1/4 the normal size.
Readout speed
Select the detector readout speed.
Slow
In slow mode, the detector readout takes about 9 seconds and introduces the lowest noise.
Fast
In fast mode, the detector readout takes about 3 seconds, but introduces more noise than slow readout mode.
Dezinger mode
Select the image dezinger mode
Normal
In normal dezinger mode, each small rotation increment is exposed twice. The two images are compared to remove cosmic rays and radioactive decay events.
None
In none dezinger mode, dezinger is not used. Each small rotation increment is exposed just once.
