File Template Dialog

The File Template dialog selects a single image or a series of images from a scan for use with the Open, Overlay, Tile, Underlay, Average, Statistics, Prev, Next and No. commands. In general, both the title to the selection dialog and the "OK" button label change to reflect the current use of the selection dialog.

Image file selection dialog.

Filter or template
This specifies the filename template. Files which match this template are listed in the Files field. It is important that the template contains question marks, if the template is to be used by Tile, Overlay, Underlay, Average, etc.
Directories
Shows the current directory name and immediate directory tree. Select a new directory by double-clicking on a directory here.
Files
Shows the files in the selected directory which match the Filter or Template. Double-click on a file to select it.
Seq start
The starting or first sequence number to be used in conjunction with the Template.
Seq incr
The increment between images. This value will be added to the starting sequence number to get the second image number. It may be negative to decrement the sequence number.
Num images
The number of images to use in any Tile, Overlay, Average or Statistics operation.
Image file
Select the filename to open or save as. This control is missing for Tile, Underlay, Overlay, and Average operations, since a series of filenames is constructed from the Template, Seq start, Seq incr and Num images controls.
Examples

Below are three examples. In each case the template, the starting sequence number, the increment and number of images are given on one line and the resulting image filenames are listing on subsequent lines.

ribosome.???? 1 1 5
ribosome.0001, ribosome.0002, ribosome.0003, ribosome.0004, ribosome.0005

ribosome?.??? 2 2 3
ribosome0.002, ribosome0.004, ribosome0.006

lyso5.??? 5 -1 3
lyso5.005, lyso5.004, lyso5.003
